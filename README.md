# ReadMeCraft

A form-based README builder for developers. Pick a template, fill in your sections, and get a professional README — no AI, no API keys, no account needed.

## Getting Started

### Prerequisites

- Node.js 18+

### Setup

```bash
# Clone the repo
git clone https://github.com/MukhtaramS/readmecraft.git
cd readmecraft

# Install dependencies
npm install

# Run the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Deploy to Vercel

1. Push to GitHub
2. Import the repo on [vercel.com](https://vercel.com)
3. Deploy — no environment variables needed

## Features

- **Project README builder** — Header, features, tech stack, installation, usage, env vars, roadmap, and more
- **Profile README builder** — GitHub stats, skill icons, social links, contribution graph
- **6 project templates** — Clean Library, API Docs, Full-Stack App, Showcase, Open Source, Startup
- **5 profile templates** — Minimalist, Stats Heavy, Creative, Developer, Social
- **Live preview** — See rendered markdown as you type
- **Copy & download** — Export as `README.md` instantly

## Tech Stack

- **Next.js 14** — App Router
- **Tailwind CSS** — Styling
- **TypeScript** — Type safety
- **react-markdown** — Live preview rendering
